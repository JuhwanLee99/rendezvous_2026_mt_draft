Overview

하나의 드래프트, 세 개의 화면

관리자가 PC에서 드래프트를 세팅·통제하고, 두 팀 대표는 휴대폰으로 순서에 맞춰 선수를 지명하며, 누구나 현황 화면에서 실시간으로 지켜보고 응원할 수 있습니다. 선수는 포지션별로 나눠 지명(팀당 같은 포지션 최대 2명)하고, 완료되면 팀별 명단을 그라운드 그림·표로 보고 이미지로 내보낼 수 있습니다. 한 곳의 변화가 모든 화면에 즉시 반영됩니다.

📡

현황 페이지 (공개)

스코어보드, 라이브 영상, 팀별 로스터(포지션 표시), 참가 선수, 접속자수, 응원 채팅, 최종 명단.

🖥️

관리자 페이지 (PC)

팀·대표·핀·포지션·선수 설정, 선픽 지정, 시작·대리지명·취소·리셋.

📱

대표자 페이지 (모바일)

팀 선택 + 4자리 핀 입장, 내 차례 알림, 포지션별 선수 지명, 내 로스터.

모바일에서도 그대로 — 실시간 + 응원

  • 실시간 동기화 — 지명 즉시 스코어·로스터·참가 선수 목록 자동 갱신
  • 지명 중 강조 — 현재 차례 팀 카드를 빨간 테두리 + "지명 중"으로 강조
  • 대형 지명 발표 팝업 — 새 선수가 뽑히면 화면 전체에 크게 발표(현장 중계용)
  • 채팅 도배 방지 — 너무 빨리 보내면 "메시지를 보낼 수 없습니다. 3초 후 다시 시도하세요."

왜 현황 화면이 중요한가

  • 현장에 못 온 부원도 휴대폰으로 실시간 관전
  • 대형 스크린에 띄워 두면 현장 중계 화면으로 활용
  • 채팅으로 모두가 함께 분위기를 만듦

  • 라이브 미러 — 우측에서 로스터·참가 선수 풀 실시간 확인

진행 단계 — 설정 잠금 + 진행 컨트롤

2드래프트 진행 통제

  • 현재 차례 표시 — 지명할 팀과 라운드(예: 청팀 R1)
  • 대리 지명 — 대표 부재 시 관리자가 대신 지명

  • 설정 잠금 — 진행 중에는 설정이 잠겨 실수 방지
Captain

대표자 페이지 — 휴대폰으로 직접 지명

각 팀 대표는 자신의 휴대폰으로 접속해 순서에 맞춰 선수를 선택합니다. 큰 버튼과 명확한 안내가 특징입니다.

① 팀 선택 (청룡/백호)
② 4자리 핀 입장
③ "지명 차례입니다!" 팝업
④ 포지션별 지명 (2/2 꽉 참 비활성)
⑤ 선수 지명 확인

팀 선택 → 핀 입장 → 내 차례 → 지명

  • 팀 선택 후 핀 입장 — 청팀/백팀을 먼저 고르고 관리자가 발급한 4자리 핀 입력(핀이 같아도 팀이 명확)
  • 내 차례 자동 팝업 — 순서가 되면 "지명 차례입니다!"가 라운드와 함께 자동 표시
  • 포지션별 지명 — 선수가 포지션 그룹으로 묶여 표시, 우리 팀이 이미 2명 채운 포지션은 「2/2 꽉 참」으로 비활성
  • 지명 확인 단계 — "○○ 선수를 지명하시겠습니까?"로 한 번 더 확인(오선택 방지)
  • 비차례 보호 — 내 차례가 아니면 선택 비활성화

현장에서 혼선 없이

  • 두 대표가 각자 자기 폰만 보면 진행 끝
  • 잘못 눌러도 확인 단계가 막고, 관리자가 즉시 되돌림
  • 순서·포지션 캡·중복은 서버에서 강제 — 차례가 아니거나 포지션이 꽉 차면 지명 자체가 막힘
How it works

드래프트 방식 — 스네이크 + 포지션

순서는 공정한 스네이크 그대로, 거기에 포지션 제한이 더해졌습니다.

1

스네이크 순서

1R 청·백 → 2R 백·청 → 3R 청·백 … 라운드마다 순서가 뒤집혀 공정합니다.

2

포지션 캡 (팀당 2명)

선수는 포지션별로 나뉘고, 한 팀은 같은 포지션을 최대 2명까지만 뽑습니다.

3

자동 스킵 & 종료

차례인데 더 못 뽑으면 그 팀은 건너뜀. 양 팀 모두 불가하면 종료.

4

서버 규칙 강제

차례·포지션 캡·중복·순번을 서버에서 검증해 차단(실시간 동기화).

Result & Export

최종 명단 — 그라운드 배치 & 이미지 내보내기

드래프트가 끝나면 팀별 명단을 표 또는 야구 그라운드 위 실제 수비 위치로 보여주고, 1:1 · 3:4 이미지로 저장해 공유할 수 있습니다. 아래는 실제 내보내기 결과 예시입니다.

  • 그라운드 배치 — 투수·포수·1루·유격·좌익… 포지션명을 실제 수비 위치로 매핑해 다이아몬드 위에 배치(내야수·외야수 묶음도 지원)
  • 표 / 그라운드 토글 — 현황·관리자 화면의 "최종 명단"에서 두 보기를 바로 전환
  • 이미지 저장 — 팀별로 1:1(정사각, SNS 피드) 또는 3:4(세로) PNG 다운로드 — 9포지션은 3:4가 여유롭습니다
  • 브랜드 일관 — 청팀=네이비, 백팀=화이트+레드. 로고·팀명·날짜 포함
